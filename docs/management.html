Simple Karma Management

Simple Karma allows viewers to rate web content (e.g. +1 or -1). This explains:
A) How to set Simple Karma's negative threshold at which web content becomes flagged, and;
B) How to locate, sort, and modify rated web content

  1. Go to WordPress
  2. Choose "Manage"
  3. Choose "Karma Options"
  4. To set the options:
    • Under "Karma Limit" enter a negative number into the "lower than:" field
      • The number you enter is the rating at which the web content will become flagged
      • When the web content's rating reaches this negative threshold, the content will be classified as flagged and the website administrator will be notified
    • Under "Over the Threshold Message" enter a phrase you would like to display when the object has reached the threshold
      • This phrase will be displayed to the user who rates the object when the objects reaches the threshold
      • When the web content's rating reaches this negative threshold, it can be deleted or changed by the admin
    • Under "bbpress Table name" enter the name of your bbpress post table.
      • If you do not plan on intergrating Simple Karma into bbpress this step is not nessecary.
      • If you do want to use simple karma in bbpress please enter the entire name of you post table, in mose cases its 'bb_posts'
  5. Choose "Update Options" to confirm your Options
  6. To access rated web content:
    • Choose "Manage"
    • Choose "Manage Simple Karma"
    • Choose a database (e.g. "wp_posts")
      • Here, administrators can modify ratings or delete flagged content
    • Rated web content can be sorted by "ascending" and "descending"
      • Choose "ascending" to view the lowest rated web content; the 50 lowest rated objects will be displayed
      • Choose "descending" to view the web content rated highest to lowest; the 50 highest rated objects will be displayed
    • To view web content in its original location on the web, choose the link "Go to Object on Site"
